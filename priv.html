Kebijakan Privasi Warung Iboo Kahel

1. Informasi Yang Dikumpulkan

  1. Nama lengkap pelanggan
  2. Alamat pengiriman
  3. Nomor telepon/WhatsApp
  4. Alamat email
  5. Riwayat transaksi

2. Penggunaan Data Pribadi

  1. Data hanya digunakan untuk keperluan pemrosesan pesanan
  2. Digunakan untuk komunikasi seputar pesanan
  3. Tidak diperjualbelikan kepada pihak ketiga
  4. Dilindungi dengan sistem keamanan digital

3. Keamanan Data

  1. Menggunakan enkripsi data standar
  2. Akses data dibatasi hanya pada staff yang berwenang
  3. Sistem keamanan website diperbarui secara berkala
  4. Data disimpan dalam server lokal terjamin keamanannya

4. Hak Pelanggan

  1. Hak untuk meminta penghapusan data
  2. Hak untuk memperbarui informasi pribadi
  3. Hak untuk tidak menerima komunikasi pemasaran
  4. Transparansi penggunaan data

5. Perubahan Kebijakan

  1. Kebijakan privasi dapat diperbarui sewaktu-waktu
  2. Perubahan akan diinformasikan melalui website
  3. Pelanggan disarankan berkala memeriksa kebijakan terbaru
